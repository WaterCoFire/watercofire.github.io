Information Security Module (Year 24/25) Notes
信息安全技术课程（2024-2025 学年）课程笔记

Be Advised 请注意

  • What is taught may vary from year to year. This note cannot always represent that.
    每届的教学内容可能不一致。此笔记不能始终代表实际教学内容。
  • Do not rely on this note. Always incorporate actual courseware for effectiveness.
    请勿依赖该笔记。请始终结合实际课件食用以发挥效果。
  • Do not share this note with non-DIICSU faculty/students, unless they are authorized.
    请勿分享给非 DIICSU 教职工/学生，除非他们经过授权。

Note Outline 笔记大纲

This note covers all 17 topics of this module：
此笔记覆盖了本课程的 17 个 topic 的全部内容：

  • Introduction to Information Security
    信息安全入门
  • Security Risk Assessment
    安全风险评估
  • Introduction to Cryptography
    密码学入门
  • Stream Ciphers
    流密码算法
  • Block Ciphers
    块密码算法
  • Hash Functions
    哈希函数
  • Passwords
    密码
  • MACs
    信息验证码
  • Web Application Vulnerabilities and Attacks
    网络应用程序漏洞和攻击
  • Public Key Cryptography
    公钥密码学
  • Digital Signature
    数字签名
  • PKI
    公钥基础设施
  • Security Protocols
    安全协议
  • SSL/TLS
  • Phishing
    网络钓鱼
  • Chaum’s Electronic Cash
    查姆电子现金
  • Bitcoin
    比特币

For many difficult points (such as the key exchange process for a complex communication protocol), WaterCoFire has done his best to explain it in layman’s terms.
对于许多难点（比如一个复杂通讯协议的密钥交换流程），WaterCoFire 尽了最大能力将它用通俗易懂的语言解释出来。

This note also covers the original questions from several quizzes of the module in Year 24/25, as well as some of the points tested on the final exam. (from memory!)
此笔记还覆盖了 2024-2025 学年本课程中若干小测的原题以及期末考试的部分考点。（都是靠回忆的喵！）
